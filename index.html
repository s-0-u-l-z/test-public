How to Set This Up (or Your Own Site)

Step 1: Get This Code on GitHub

If you haven't already, you need a GitHub account and a repository (a place to store your code).

  • Create a new repository on GitHub. You can name it anything you like, for example, my-awesome-page.
  • Download this HTML file (or copy its content into a new file named index.html).
  • Add this index.html file to your new repository and push it to GitHub.
    git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY_NAME.git cd YOUR_REPOSITORY_NAME # (Add your index.html file here) git add index.html git commit -m "Add initial page template" git push origin main # or master, depending on your default branch

Step 2: Enable GitHub Pages

Now, tell GitHub to publish your site:

  • Go to your repository on GitHub.
  • Click on the Settings tab.
  • In the left sidebar, scroll down and click on Pages (under "Code and automation").
  • Under "Build and deployment", for the "Source" option, select Deploy from a branch.
  • Then, select the branch your index.html file is on (usually main or master).
  • Ensure the folder is set to /(root).
  • Click Save.

Step 3: Visit Your Live Site!

GitHub Pages will build and deploy your site. This might take a minute or two.

  • Once it's ready, the Pages settings page will display a link to your live site. It will look something like:
    https://YOUR_USERNAME.github.io/YOUR_REPOSITORY_NAME/
  • Click the link, and you should see this very page!
  • Any changes you push to your index.html file on the selected branch will automatically update your live site.
